📌 Overview

This project is an end-to-end Machine Learning application that predicts book ratings based on various input features such as title, author, genre, and user-related attributes. The goal is to build a complete ML pipeline including data preprocessing, feature engineering, model training, evaluation, and prediction.

This project demonstrates strong understanding of data science workflow and production-ready ML structure.
